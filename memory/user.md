# User Profile

## Background

Building a casual mobile puzzle game ("Guess the Dots") in Godot 4 targeting Android.
The game is a Mastermind/Wordle-style color-pattern guessing game. The user wants to
monetize with AdMob and grow a loyal, returning player base.

Goal is to make the game feel polished, futuristic, and addictive — with a 100-level
Campaign Mode as the primary long-term engagement driver. Daily Challenge was removed
in favour of Campaign to give players a clearer progression arc.

## Technical Level

- Using Godot 4 (GDScript)
- Has AdMob integration already working
- Comfortable with scene/script structure — existing code is clean and well-organized
- Building for Android mobile (portrait orientation)

## Working Style

- Likes ambitious, big-picture plans broken into phases
- Wants brainstorming + comprehensive planning documents (plan.md style)
- Wants futuristic/cyberpunk visual direction
- Interested in both design (UX, game feel) and implementation
- OK with creating new files when necessary for major features
