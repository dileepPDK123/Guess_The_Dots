A · Count-Only (All Modes)

Pips only tell you the total count of exact + misplaced. You figure out which dots are correct by reasoning across guesses. True Mastermind.

TWO FEEDBACK MODES
CLASSIC / HARD / ZEN / BLITZ
Pips = count only. You reason it out. ✓ Authentic
EASY MODE (new) — for beginners
Each dot gets a colored ring. Green ring = this dot is exact. Yellow = this color is in seq, wrong slot.
Easy mode is optional / selectable. Default is count-only.

B · Mode Split

All main modes use count-only (authentic). A new "Easy" mode shows per-dot feedback for beginners or casual play.

My recommendation: Option B. It keeps the real game honest while giving new players a path in. The color tracker already helps experienced players — Easy mode is for the rest.
