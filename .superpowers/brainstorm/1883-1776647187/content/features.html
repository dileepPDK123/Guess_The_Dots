What new features should we add?

Researched from Wordle, Mastermind apps, Lingo, and top mobile puzzle games. Select all that interest you — we'll scope later.

♿

Color-Blind Mode

Add shapes inside dots (circle, square, triangle, star) so the game works without color perception. Very common in top puzzle games.

🔄

Flip Animation on Submit

Each dot in the submitted row flips like Wordle to reveal its feedback one by one. Satisfying, iconic, builds suspense.

🎯

Color Elimination Tracker

A small row below the palette that marks each color as: confirmed in sequence, confirmed absent, or unknown. Helps strategy.

🔗

Create & Share a Puzzle

Set your own secret sequence and share a link/code for a friend to guess. Like Wordle's custom word links. Very shareable.

📤

Share Result Grid

Post-game share as emoji grid (already partially built). Polish it with score, mode, streak, and app link for social media.

🧠

Smart Hint: Eliminate a Color

Instead of revealing a slot, hint tells you one color that's definitely NOT in the sequence. More strategic, less spoiling.

⏱️

Blitz Revamp: Visible Timer Ring

Replace the text countdown with a circular ring that depletes around the board. Instantly readable, dramatic when nearly empty.

🏆

Weekly Leaderboard

Daily challenge rankings by fewest guesses, fastest time. Resets weekly. Drives daily engagement.

🎉

Win Celebration: Dot Burst

On win, the correct dots burst outward from the board with a confetti-like scatter animation using the actual dot colors.

🔒

Hard Mode Visual Lock

In Hard mode, visually lock confirmed exact slots with a padlock icon and a satisfying click animation when they lock.

Select as many as you like. The most impactful from research: color-blind mode, flip animation, color tracker, and share grid — these appear in every successful Mastermind-style game.
